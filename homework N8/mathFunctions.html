Math Functions

The math functions allow CSS numeric values to be written as mathematical expressions. calc() A math function that allows basic arithmetic to be performed on numerical CSS values. clamp() A comparison function that takes a minimum, central, and maximum value and represents its central calculation. max() A comparison function that represents the largest of a list of values. min() A comparison function that represents the smallest of a list of values. abs() Experimental Takes a calculation and returns the absolute value.
